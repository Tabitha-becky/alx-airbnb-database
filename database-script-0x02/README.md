# Airbnb Database Extended Seed Data

This folder contains an extended dataset to populate the Airbnb database with realistic sample data.

## Tables Seeded

1. **Users** – 10 sample hosts and guests.
2. **Properties** – 8 sample listings with different cities, prices, and max guests.
3. **Bookings** – 12 sample reservations linking guests and properties.
4. **Reviews** – 10 sample reviews with ratings and comments.
5. **Payments** – 12 payment records linked to bookings with different payment methods.

## Instructions

1. Make sure the database schema (`schema.sql`) is created first.
2. Run this SQL script (`seed.sql`) to populate the tables.
3. Adjust dates, prices, or names if needed for testing.
